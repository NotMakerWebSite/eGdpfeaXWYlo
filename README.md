## SSM宠物交易系统

> [点我下载源码或在线试用](https://www.notmaker.com/detail/13f08d6e823f4a6ba5002329ab709e0e/ghb20250812) 

> 支持定制、讲解、修改、远程调试

### 一、相关技术
- 后端：`Java`、`JavaWeb` / `SSM`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript`
- 数据库：`MySQL`

### 二、相关软件
- `IDEA`
- `Navicat`

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 宠物用品
4. 宠物商店
5. 宠物领养
6. 宠物挂失
7. 宠物论坛
8. 宠物咨询
9. 个人中心
10. 购物车
11. 联系客服


**管理员功能：**
1. 登录
2. 宠物分类管理
3. 商品分类管理
4. 宠物用品管理
5. 宠物商店管理
6. 宠物领养管理
7. 用户管理
8. 宠物寄存管理
9. 用户领养管理
10. 宠物挂失管理
11. 论坛管理
12. 管理员管理
13. 系统管理
14. 订单管理
